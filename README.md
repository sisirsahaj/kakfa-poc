# Kafka
  1. `cd kafka`
  2. `docker-compose down`
  3. `docker-compose up`

# producer

To produce messages run 


``` python producer.py ```


# consumer

Step 1: ``` cd consumer-webapp && node server.js ```

Step 2: ``` cd kafka-websocker-consumer && npm start ```
