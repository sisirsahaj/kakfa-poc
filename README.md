# Kafka
  1. `cd kafka`
  2. `docker-compose down`
  3. `docker-compose up`


# consumer

Step 1: ``` cd consumer-webapp && node server.js ```

Step 2: ``` cd react-app && npm start ```

# producer

To produce messages run 

``` pip install -r requirements.txt ```
``` python producer.py ```
